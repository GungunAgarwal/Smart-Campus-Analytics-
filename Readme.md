# Smart Campus Operations Analytics System

## Project Overview
This project analyzes campus operations using SQL, Python, Excel and Power BI. It provides insights into student records, attendance, fees, library usage, hostel occupancy and IT service tickets through an interactive dashboard.

## Technologies Used
- Python
- SQL Server
- Excel
- Power BI

## Features
- Student Management
- Attendance Analysis
- Fee Collection Analysis
- Library Usage
- Hostel Occupancy
- IT Service Desk Analysis
- Interactive Power BI Dashboard

## Dataset
Synthetic datasets generated using Python.

## Dashboard KPIs
- Total Students
- Total Departments
- Attendance %
- Fee Collection
- Library Books Issued
- Hostel Occupancy
- IT Tickets

## Project Structure
Python → Dataset → SQL Server → Power BI Dashboard
